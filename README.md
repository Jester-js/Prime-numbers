# Prime numbers
> Sieve of Eratosthenes

-----------------

## About:
Search prime numbers based on algorithm called "Sieve of Eratosthenes".

------------------

## How to use:
Enter limit number and click on button "GET PRIME NUMBERS".

------------------

## P.S:
Do not recommend to enter very big numbers.
