#PRIME NUMBER
###with sieve of Eratosthenes
-----------------

###About:
Пошук простих чисел, побудований на алгоритмі, що називається "Решето Ератосфена". 

------------------
###How to use:
Просто введіть число, яке буде межею для пошуку. 

#####P.S: Не рекомендую вводити дуже великі числа)
